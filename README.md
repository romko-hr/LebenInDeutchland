# Leben in Deutschland Flashcards

This repository contains a static study site for the `Leben in Deutschland` exam vocabulary.

## GitHub Pages Setup

1. Push this repository to GitHub.
2. Open `Settings -> Pages`.
3. In `Build and deployment`, choose `Deploy from a branch`.
4. Select your main branch (`main` or the branch you use).
5. Select the folder `/ (root)`.
6. Save.

GitHub Pages will publish the root [index.html](/Users/romko/LebenInDeutchland/index.html), which redirects to the flashcard app in [web/index.html](/Users/romko/LebenInDeutchland/web/index.html).
